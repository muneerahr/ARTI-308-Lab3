# Exploratory Data Analysis (EDA) 

## Project Overview
This project performs Exploratory Data Analysis (EDA) on a demographic subset of the National Health and Nutrition Examination Survey (NHANES) dataset 
Although NHANES includes extensive medical, laboratory, and health-related data, this analysis focuses only on demographic and socioeconomic variables.

## Dataset Source
Source:
NHANES – National Health and Nutrition Examination Survey  
The dataset was accessed via Kaggle.

## Dataset Description
This analysis uses the demographic file from the NHANES dataset, which contains variables such as:

- **RIDAGEYR** → Age (in years)
- **RIAGENDR** → Gender
- **DMDHHSIZ** → Household Size
- **INDFMPIR** → Family Income to Poverty Ratio

A data dictionary is available with the dataset to explain variable definitions.

## Analysis Performed
The following exploratory analyses were conducted:

- Age distribution (Histogram)
- Gender distribution analysis
- Age vs Household Size (Scatter plot)
- Income Ratio distribution
- Income Ratio trend by age
- Correlation analysis between selected variables
